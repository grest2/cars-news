# cars-news
<sup>Данное приложение содержит в себе новостную ленту для с использованием api - autodoc, при нажатии на новость открывается страница новости, при долгом нажатии на новость, она будет выделена.</sup> 
**Оно выполнено для пользования на iPhone и iPad**
